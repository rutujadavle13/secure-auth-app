## Secure Auth App

A full-stack web application that provides secure user registration, login, and protected routes using Node.js, Express, and MongoDB.

## Features

- User Registration & Login
- Password hashing with bcrypt
- Session-based authentication
- Protected dashboard
- Logout functionality

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB (Mongoose)
